Database Docs (RP)¶

class database.Database¶

Database abstraction class for handling all access to Database.

add_user(username: str, firstname: str, lastname: str, email: str, password: str)¶

This function will add a user to the database.

check_create_user_table()¶

This function will check if a user table has been created. If not, create the table.

check_user(username: str)¶

This function gets and checks if a certain username exists in the User table, before they are connected.

close()¶

This function commits database changes, closes the connection and sets the connection flag to false.

connect()¶

This function establishes a connection to an SQLite database. If file doesn’t exist it will be created. Detect_types is used for mapping Python types to database types.

get_password(username: str)¶

This function gets the password for a specific user from the User table.

get_user(username: str)¶

This function gets and returns a user and his/her details, using their unique username as the argument.
