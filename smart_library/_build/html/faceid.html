Face ID Docs (RP)¶

class faceid.FaceID¶

This class handles the creation, encoding and recognization of user images.

encode_images(username: str)¶

This method will extract the embeddings from the faces in the photos taken and stored before. An encodings.pickle file will be generated at the conclusion of this method. The username is taken as an argument to locate the exact folder of images that are to be encoded.

recognize_user()¶

This method will loop over the known faces and embeddings, and look to identify that user in real time.

register_user(username: str)¶

This method will register a user by firstly creating a unique folder under their username (taken as an argument) they choose to be idenitified as while using the application, then followed by their pictures being taken via the webcam to be stored in that folder.
