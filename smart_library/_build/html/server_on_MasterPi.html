Server On Master Pi Docs¶

class server_on_MasterPi.Server¶

This class will create a server on the Master Pi that will continuously listen for and accept client socket connections.

print_server_info()¶

This method will print out a confirmation message that the server is now running on the host and port number it’s been assigned to.

server_listening()¶

This method is where the server listens for the client connections.
