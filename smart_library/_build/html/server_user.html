Server User Docs (MP)¶

class server_user.ServerUser(lmsuserid: int, username: str, name: str)¶

This class deals with a user on the library server, taking in the arguments of their user id on the system, their username and their actual name.

get_lmsuserid()¶

Gets and returns the user id of a library user.

get_name()¶

Gets and returns the actual name of a library user.

get_username()¶

Gets and returns the username of a library user.
